Future Work

In future versions of the BDI model we will introduce automatic Plan compilation using the whole database of services offered by all the agents in the platform. In order to do that, the platform will include a deliberative agent able to make searches through all the available services and compose a Plan on-the-fly whenever the set of pre-compiled Plans of an agent is not enough to achieve the desired Goal. But that, my friends, is another story
